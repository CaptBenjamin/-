## 关于 CaptB 点名器名单文件的生成
1. 制作一个 txt 名单文件，第一行为该名单中的人数。从第二行开始每行一个人名，无需空行。
2. 进入 CBRC 导入名单，文件类型筛选选择 *.* ，完成导入。
3. 点击 CBRC 内的导出名单即可生成名单文件。
